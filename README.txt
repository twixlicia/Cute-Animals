README.txt

File Structure:

1. index.html
Purpose: Main HTML file containing the structure of the website.
Relative Path: index.html

2. info.html
Purpose: HTML file containing information about cute animals and a subscription form.
Relative Path: info.html

3. pictures.html
Purpose: HTML file displaying a grid of cute animal pictures.
Relative Path: pictures.html

4. styles.css
Purpose: CSS file containing styles for the website.
Relative Path: styles.css

5. script.js
Purpose: JavaScript file containing scripts for interactive features.
Relative Path: script.js

6. info.js
Purpose: JavaScript file containing scripts for the subscription form in info.html.
Relative Path: info.js


Functionality and Dependencies: 

1. The website utilizes HTML, CSS, and JavaScript for its functionality.

2. JavaScript is used for interactive features such as changing hero images and toggling the burger menu.

3. The website is responsive and adjusts its layout using CSS media queries for different screen sizes.
